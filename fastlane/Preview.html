Description
Ever arrived to an empty bike sharing station? Open Bicis and glance detailed bike sharing availability predictions next to the daily history.

New predictions are created daily at midnight. Custom tailored predictions by a neural network using previous data. Compare the actual behavior of a station with the predicted availability.

Available for Madrid, Bilbao, New York and more to come!
Changelog
Wow, this is a big update full of goodies.

The neural networks that power this app and generate predictions have been updated. This is not done in the app but on a server but I wanted to let you know you are getting more and better predictions. I've used the most recent data so the quality should improve.

I've improved how the graph looks, lines should be smoother and more pleasant now. Giving it a fresher look.

The insights view has been completely redesigned from the ground up. Now it shows even more data including the graph so you can compare it against the data that it's shown in this screen.

Enjoy!

Description
¿Has llegado alguna vez a una estación de bicicletas y estaba vacía? Abre Bicis y obtén información detallada sobre la predicción de disponibilidad durante el día junto con el historial. Compara ambas para ver el comportamiento y decide tu medio de transporte.

Cada día se crean nuevas predicciones. Cada una es personalizada para cada estación por una red neuronal utilizando datos de días anteriores.

Disponible para Madrid, Bilbao, Nueva York y más por llegar.

Changelog
Wow, actualización cargada de novedades.

Las redes neuronales que realizan las predicciones han sido actualizadas. Esto no forma parte de la app sino del servidor pero quería informar de que ya se puede disfrutar de mejores predicciones. Todo esto sin necesidad de actualizar la app. He utilizado los datos más recientes para que las predicciones sean lo más acertadas posibles.

El gráfico de las predicciones ahora es más bonito, las líneas son más suaves y el gráfico es más agradable de ver.

He rediseñado la vista desbloqueable de detalles de una estación completamente, ahora muestra más detalles y el gráfico de predicciones así puedes comparar el comportamiento con los datos.
