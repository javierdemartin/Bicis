Description
Ever arrived to an empty bike sharing station? Open Bicis and glance detailed bike sharing availability predictions next to the daily history.

New predictions are created daily at midnight. Custom tailored predictions by a neural network using previous data. Compare the actual behavior of a station with the predicted availability.

Available for Madrid, Bilbao, New York and more to come!

Description
¿Has llegado alguna vez a una estación de bicicletas y estaba vacía? Abre Bicis y obtén información detallada sobre la predicción de disponibilidad durante el día junto con el historial. Compara ambas para ver el comportamiento y decide tu medio de transporte.

Cada día se crean nuevas predicciones. Cada una es personalizada para cada estación por una red neuronal utilizando datos de días anteriores.

Disponible para Madrid, Bilbao, Nueva York y más por llegar.
